技术架构

三层存证架构

🗂️
版本层
Git
对 Agent memories 文件夹进行版本管理，每次交易决策前自动 commit
# 交易前自动提交
git add memories/
git commit -m "pre-trade"
🌐
存储层
IPFS
记忆快照存储到 IPFS，获得内容寻址的 CID，确保数据永久可访问
# 上传到 IPFS
CID = ipfs_pin(snapshot)
# → bafy...xyz
◆
证明层
Monad
将 Git Hash + IPFS CID + Memory Root 记录到 Monad 链上合约
// Solidity 合约
recordMemory(
  gitHash, cid, root
)
